## 📂 Projects

### 🔍 Credit Card Fraud Detection

**Problem:** Detect fraudulent transactions in highly imbalanced data

**Key Work:**

* Time-based feature extraction (hour, weekday, patterns)
* User behavior modeling (transaction history, averages)
* Distance-based anomaly detection
* Handling class imbalance with weighted models
* Threshold tuning for recall optimization

**Impact:**

* Improved fraud detection recall significantly
* Maintained high precision
* Simulated real-world fraud detection pipeline
